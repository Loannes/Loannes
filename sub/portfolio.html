Insight

Instagram scraping, tracking, content creation, management system
 
Language :
Ruby On Rails

인사이트는 인스타그램을 크롤링(혹은 스크래핑) 하여 특정 목적을 가진 정보를 추출, 추적, 관리하는 시스템이다.
 
기능 1. 특정 타겟팅된 포스팅을 읽어들여 원하는 데이터를 추출
기능 2. 추출된 정보에서 유용한 정보만 선별
기능 3. 선별된 정보를 주기적으로 데이터의 변화를 트래킹
기능 4. 선별된 정보를 기반으로 컨텐츠를 생성
